OpenAI-kompatible Schnittstelle für Home Assistant – Docker-ready, gehärtet, nur API-Key rein und los.

OpenAI compatible FastAPI · Python 3.11+ Rate limited

What it does

  • Slim, hardened AI gateway between Home Assistant and any OpenAI-compatible API.
  • Input validation, rate limiting & auth out of the box.
  • Route Assist, automations & scripts through your preferred LLM.
★ GitHub 🔒 Security

Quick Start

  • Copy .env.example to .env and set your API key.
  • Run docker compose up -d.
  • Point HA Assist or rest_command at http://hai-haaii:8787.
cp .env.example .env # set OPENAI_API_KEY etc. docker compose up -d
